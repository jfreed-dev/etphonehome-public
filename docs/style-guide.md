## Palette

- `--et-bg-900` #0B1020
- `--et-bg-800` #111B2F
- `--et-bg-700` #14263B
- `--et-accent-cyan` #2DE2E6

- `--et-accent-amber` #FFB454

- `--et-text-300` #B8D9E6

## Typography

- Display: Space Grotesk / Futura
- Mono: Fira Mono / SFMono-Regular
- Fallback: sans-serif

## Components

- Panel: dark surface with cyan border and soft radius
- Chips: uppercase tags with cyan tint
- CTA: cyan to green gradient button
